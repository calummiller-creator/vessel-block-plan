RBP Activities Manager — Guide
Activity levels
Standalone A single activity with no sub-activities. Has its own vessel, dates, and location.
L1 (Campaign) A campaign header row. Contains the vessel, SPA, grouping, and window dates for the whole campaign. Its From/To dates auto-update from its L2 activities.
L2 (Sub-activity) A sequential activity within a campaign. Hidden from the main table — view them by clicking the L2 count button on an L1 row.
Main table
Region selector Switch between North Sea, M&S, and Egypt. Reloads data from SharePoint.
Tabs Filter by All, Campaigns only, or Standalone only.
Filters Click any filter dropdown to multi-select values. Active filters show a blue count badge. "Clear all" resets everything.
Sorting Click column headers (Title, Area, Asset, etc.) to sort. Click again to reverse.
Show completed By default, Completed/Cancelled activities are hidden. Toggle this to show them.
Copy to Excel Copies the currently visible table to clipboard for pasting into Excel.
Edit button Opens the edit form for a Standalone or L1 row.
Convert to L1 On standalone rows, click the arrow button to convert it into a campaign with L2 sub-activities.
L2 sub-activities
View L2s Click the count button (e.g. "4 →") on an L1 row to open the full-page L2 editor.
Date chaining Set the first activity's From date, then enter durations. Each subsequent activity starts the day after the previous one ends.
Override dates You can manually change any From or To date. Subsequent rows will re-chain from that point.
Defaults from L1 New L2 rows inherit Vessel, SPA, Grouping, Org, Status, and P/E from the L1 campaign header. Override any of these per row if needed — differences show in blue.
Blank Area/Asset/Field Port calls and transits can leave these blank.
SPA search Type 3+ characters to search the global address book. Select a result to set the SPA.
Paste from Excel
Download template Click "Download Excel template" to get an Excel file with dropdown validations for all choice columns.
Fill in template Enter activities in the template. Only Title and Duration are required — everything else defaults from the L1.
Paste Select your data rows (not the header), copy, click "Paste from Excel" in the L2 editor, paste into the text box, and click "Import rows".
Column order Title, SubType, Area, Asset, Field, Duration, Vessel, MMSI, SPA, Grouping, Org, Status, P/E
Copy to Excel
Main table Click "Copy to Excel" in the header to copy the visible filtered data.
L2 table Click "Copy to Excel" in the L2 editor to copy all sub-activities. Useful for making changes in Excel and pasting back.
Saving
All changes save to SharePoint via Power Automate. The HTML contains no data — everything loads and saves in real time.
L1 dates auto-update When you save L2 activities, the L1 campaign's From and To dates automatically update to match the earliest and latest L2 dates.
